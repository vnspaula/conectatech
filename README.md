# ConectaTech

Frontend da plataforma ConectaTech.

## Tecnologias

- React
- TypeScript
- Vite
- Firebase

## Como executar

```bash
npm install
npm run dev
```

## Status

🚧 Em desenvolvimento.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
